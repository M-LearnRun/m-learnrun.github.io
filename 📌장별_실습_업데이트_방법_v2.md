# 📌 장별 실습 데이터 업데이트 방법 (v2 - 간소화 버전)

## 🎯 한눈에 보는 업데이트 프로세스

```
1. 데이터 파일 추가 → data/chapter4/ 폴더에 복사
2. chapter.html 파일 열기 → 텍스트 에디터로 열기
3. 해당 챕터 찾기 → '4': { ... } 부분 찾기
4. 정보 수정 → 아래 예시대로 수정
5. 저장 후 테스트 → 웹페이지에서 확인
```

## ⚠️ 주의사항

### 1. preview 필드 제거됨
```javascript
// ❌ 구버전 (더 이상 사용 안 함)
{
    title: '제목',
    platform: 'chatgpt',
    desc: '설명',
    preview: '프롬프트 미리보기...',  // ← 제거!
    link: 'https://...'
}

// ✅ 신버전 (간소화)
{
    title: '제목',
    platform: 'chatgpt',
    desc: '설명',
    link: 'https://...'
}
```

### 2. platform 값 정확히 입력
```javascript
platform: 'chatgpt'      // ✅ 소문자
platform: 'ChatGPT'      // ❌ 대문자 X
platform: 'chat-gpt'     // ❌ 하이픈 X
```

## 📋 체크리스트

매번 업데이트할 때 확인하세요:

- [ ] 1. 데이터 파일을 `data/chapter숫자/` 폴더에 복사했나?
- [ ] 2. `chapter.html` 파일을 열었나?
- [ ] 3. 올바른 챕터 번호를 찾았나? (예: `'4': {`)
- [ ] 4. `data: []` 배열에 파일 정보를 추가했나?
- [ ] 5. `prompts: []` 배열에 간소화된 포맷으로 추가했나?
- [ ] 6. `preview` 필드를 제거했나? (v2에서는 불필요)
- [ ] 7. `platform` 값이 올바른가? (chatgpt, claude 등)
- [ ] 8. 파일을 저장했나?
- [ ] 9. 웹브라우저에서 테스트했나?
- [ ] 10. AI 로고가 제대로 표시되는가?

## 🚀 빠른 테스트 방법

수정 후 바로 확인:

1. `chapter.html` 저장
2. 웹브라우저에서 `index.html` 열기
3. "장별 실습 제공" 클릭
4. 해당 챕터 클릭 (예: Chapter 4)
5. **프롬프트 카드 확인**:
   - ✅ AI 로고가 표시되는가?
   - ✅ 제목이 정확한가?
   - ✅ 설명이 보이는가?
   - ✅ "프롬프트 보기" 버튼이 작동하는가?
6. 프롬프트 링크 클릭 → 페이지 이동 확인
