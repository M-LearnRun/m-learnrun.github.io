# 📌 장별 실습 데이터 업데이트 방법 (초간단 가이드)

## 🎯 한눈에 보는 업데이트 프로세스

```
1. 데이터 파일 추가 → data/chapter4/ 폴더에 복사
2. chapter.html 파일 열기 → 텍스트 에디터로 열기
3. 해당 챕터 찾기 → '4': { ... } 부분 찾기
4. 정보 수정 → 아래 예시대로 수정
5. 저장 후 테스트 → 웹페이지에서 확인
```

## 🔍 정확한 수정 위치 찾는 법

### 방법 1: 검색 기능 사용 (추천)
1. `chapter.html` 파일 열기
2. `Ctrl + F` (찾기)
3. `'4': {` 검색 → Chapter 4 부분으로 바로 이동!

### 방법 2: 줄 번호로 찾기
- VSCode나 Notepad++ 사용
- 약 **260줄~380줄** 사이에 `chapters` 객체 있음
- 각 챕터별로 구분되어 있음

## ⚠️ 주의사항

### 1. 쉼표(,) 주의!
```javascript
// ❌ 틀린 예 (마지막 항목 뒤에도 쉼표)
data: [
    { name: 'file1.csv', desc: '설명1', file: 'file1.csv' },
    { name: 'file2.csv', desc: '설명2', file: 'file2.csv' },  // ← 마지막이면 쉼표 제거
]

// ✅ 맞는 예
data: [
    { name: 'file1.csv', desc: '설명1', file: 'file1.csv' },
    { name: 'file2.csv', desc: '설명2', file: 'file2.csv' }   // ← 쉼표 없음
]
```

### 2. 따옴표 주의!
- 작은따옴표 `'` 또는 큰따옴표 `"` 일관되게 사용
- 문자열은 반드시 따옴표로 감싸기

### 3. 파일명 정확히!
```javascript
file: 'training_data.csv'  // ← 실제 파일명과 똑같이!
```

## 🚀 빠른 테스트 방법

수정 후 바로 확인:

1. `chapter.html` 저장
2. 웹브라우저에서 `index.html` 열기
3. "장별 실습 제공" 클릭
4. 해당 챕터 클릭 (예: Chapter 4)
5. 데이터 목록 확인
6. 다운로드 버튼 클릭 → 파일 다운로드되는지 확인
7. 프롬프트 링크 클릭 → 페이지 이동 확인

## 💡 자주 묻는 질문

### Q: 모든 챕터를 한번에 수정해야 하나요?
A: 아니요! 필요한 챕터만 하나씩 수정하시면 됩니다.

### Q: 프롬프트가 없는 챕터는요?
A: `prompts: []` 그대로 두면 "곧 업데이트 예정입니다" 표시됩니다.

### Q: 데이터가 없는 챕터는요?
A: `data: []` 그대로 두면 "곧 업데이트 예정입니다" 표시됩니다.

### Q: 파일 개수 제한이 있나요?
A: 없습니다! 원하는 만큼 추가하세요.

## 📞 문제 발생 시

### 페이지가 안 보이거나 에러 발생
1. 브라우저 F12 → Console 탭 확인
2. 에러 메시지에서 줄 번호 확인
3. 쉼표, 따옴표, 중괄호 `{}` 확인
4. 원본 백업 파일로 복구 후 다시 시도
