## 챕터별 데이터 추가 방법

### 1️⃣ 데이터 파일 준비
각 챕터 폴더에 실습에 필요한 데이터 파일을 추가합니다.

```bash
# 예시: Chapter 4에 데이터 추가
data/chapter4/training_data.csv
data/chapter4/test_data.csv
data/chapter4/model_training.py
data/chapter4/feature_engineering.py
```

### 2️⃣ chapter.html 파일 수정

`chapter.html` 파일의 JavaScript 부분에서 해당 챕터의 데이터 정보를 업데이트합니다:

```javascript
'4': {
    title: '머신러닝 모델 구축',
    desc: '회귀, 분류 모델을 활용한 주가 예측 시스템을 구축합니다.',
    data: [
        {
            name: 'training_data.csv',
            desc: '모델 학습용 데이터',
            file: 'training_data.csv'
        },
        {
            name: 'test_data.csv',
            desc: '모델 테스트 데이터',
            file: 'test_data.csv'
        }
        // 추가 데이터...
    ],
    prompts: [...]
}
```

## AI 프롬프트 링크 추가 방법

### ChatGPT 프롬프트 추가

1. ChatGPT에서 대화 공유 링크 생성
2. `chapter.html`의 해당 챕터에 프롬프트 정보 추가:

```javascript
prompts: [
    {
        title: '주가 예측 모델 설계',
        platform: 'chatgpt',  // 'chatgpt' 또는 'claude'
        desc: '선형회귀, 랜덤포레스트, XGBoost를 비교합니다.',
        preview: '주가 예측을 위한 머신러닝 모델을 설계하려고 합니다...',
        link: 'https://chatgpt.com/share/YOUR-SHARE-LINK'
    }
]
```

### Claude 프롬프트 추가

## 챕터 정보 수정

### chapters.html 수정

챕터 목록 페이지의 각 챕터 카드에서 다음 정보를 수정할 수 있습니다:

- 챕터 제목
- 설명
- 데이터 개수
- 프롬프트 개수

```html

    </p>
    <div class="chapter-meta">
        <span class="meta-item">📊 데이터: 6개</span>

    </div>

```

## 전체 업데이트 워크플로우

### 새로운 챕터 데이터 추가 시:

1. **데이터 파일 추가**
   ```bash
   # 해당 챕터 폴더에 파일 복사
   cp 새데이터.csv data/chapter4/
   ```

2. **chapter.html 수정**
   - JavaScript의 `chapters` 객체에서 해당 챕터 정보 업데이트
   - `data` 배열에 새 파일 정보 추가
   - `prompts` 배열에 AI 프롬프트 링크 추가

3. **chapters.html 수정** (선택사항)
   - 데이터 개수, 프롬프트 개수 업데이트

4. **테스트**
   - 웹사이트 열어서 챕터 페이지 확인
   - 다운로드 링크 작동 확인
   - AI 프롬프트 링크 확인

## 프롬프트 링크 생성 방법

### ChatGPT
1. 대화 완료 후 우측 상단 공유 버튼 클릭
2. "Create share link" 선택
3. 생성된 링크 복사 (예: `https://chatgpt.com/share/...`)

### Claude
1. 대화 완료 후 공유 아이콘 클릭
2. "Share conversation" 선택
3. 생성된 링크 복사 (예: `https://claude.ai/share/...`)

## 자동화 팁

모든 챕터 데이터가 준비되면 `update_data.bat`를 실행하여:
- 가격 데이터 (pd1_ohlcv) 자동 업데이트
- ZIP 파일 자동 생성

챕터별 데이터는 수동으로 관리하지만, 필요시 자동화 스크립트를 추가로 만들 수 있습니다.

## 예제: Chapter 4 완전 설정

```javascript
// chapter.html 내부
'4': {
    title: '머신러닝 모델 구축',
    desc: '회귀, 분류 모델을 활용한 주가 예측 시스템을 구축합니다.',
    data: [
        { name: 'training_data.csv', desc: '모델 학습용 데이터', file: 'training_data.csv' },
        { name: 'test_data.csv', desc: '모델 테스트 데이터', file: 'test_data.csv' },
        { name: 'feature_engineering.py', desc: '피처 엔지니어링 코드', file: 'feature_engineering.py' },
        { name: 'model_training.py', desc: '모델 학습 스크립트', file: 'model_training.py' }
    ],
    prompts: [
        {
            title: '주가 예측 모델 설계',
            platform: 'chatgpt',
            desc: '선형회귀, 랜덤포레스트, XGBoost를 비교하여 최적 모델을 선택합니다.',
            preview: '주가 예측을 위한 머신러닝 모델을 설계하려고 합니다. 어떤 모델이 적합하고, 각 모델의 장단점은 무엇인가요?',
            link: 'https://chatgpt.com/share/68afadd0-c5c8-800b-8ed0-ccc45f98fabc'
        },
        {
            title: '피처 중요도 분석',
            platform: 'claude',
            desc: '모델 성능에 영향을 미치는 주요 특성을 파악합니다.',
            preview: '학습된 모델에서 feature importance를 분석하고, 가장 중요한 변수 5개를 시각화해주세요.',
            link: 'https://claude.ai/share/example1'
        }
    ]
}
```

이제 독자들이 Chapter 4 페이지에서:
- 4개의 데이터 파일 다운로드 가능
- 2개의 AI 프롬프트 링크로 직접 이동 가능
- 프롬프트 내용을 클립보드에 복사 가능
